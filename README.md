# 在自定义 Toolbar 和 Tabbar 之间切换显示

本工程为教程文章的示例项目，[教程参考在这里](https://melerpaine.com/2017/11/27/如何在自定义 Tool Bar 和 Tab Bar 之间切换显示/)。

本工程自定义了一个 Toolbar，并模拟 Tabbar 的布局和显示效果，用于在页面的两种状态下无缝切换显示。
